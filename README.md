# Vue3项目模板

## 🐶环境

node 16+

技术栈采用:

Vue3+TypeScript+Pinia+Vite+vue-router+Element Plus

包管理采用pnpm

```shell
npm install -g pnpm
pnpm install
```

## 🐯代码提交规范

husky 是一个 Git Hook 工具，可以让你在 Git 提交过程中自动运行脚本。

当commit的时候会执行脚本，脚本会校验commit message是否符合规范。

严格安装如下规范:

| 类型     | 描述                     |
| -------- | ------------------------ |
| feat     | 新功能                   |
| fix      | 修补 bug                 |
| docs     | 文档                     |
| style    | 格式方面的优化           |
| refactor | 重构                     |
| test     | 测试                     |
| chore    | 构建过程或辅助工具的变动 |

比如修复一个bug 必须是

fix: 修复bug

中间有:后面有空格

| i18n Ally                     | 国际化智能提示                                   |
| Stylelint                     | Css 格式化                                       |

| ESLint                        | 脚本代码检查                                     |
| DotENV                        | env 文件高亮                                     |
